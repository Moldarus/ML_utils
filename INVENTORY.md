# Опись репозитория для государственного экзамена

## Владелец: Кафтанат М. Л.
## Специальность: Информатика и вычислительная техника

## Структура репозитория:

### 1. `utils.py` (основной модуль)
- **Назначение:** Набор утилит для решения задач анализа данных и машинного обучения
- **Объём:** ~600 строк кода
- **Авторство:** 100% собственная реализация
- **Функции:**
  - EDA: `quick_eda()`, `plot_distributions()`, `plot_correlation_matrix()`
  - Предобработка: `handle_missing_values()`, `encode_categorical()`, `scale_features()`
  - Кластеризация: `kmeans_elbow_method()`, `perform_clustering()`
  - Классификация: `train_classification_models()`
  - Регрессия: `train_regression_models()`
  - Аномалии: `detect_anomalies_iqr()`, `detect_anomalies_zscore()`
  - Визуализация: `quick_visualization_report()`
  - Генерация данных: `generate_sample_data()`

### 2. `exam_examples.py` (примеры использования)
- **Назначение:** Демонстрация работы всех утилит
- **Авторство:** Собственные примеры

### 3. `requirements.txt`
- **Назначение:** Список зависимостей
- **Особенность:** Все библиотеки имеют открытые лицензии

### 4. `main.py`
- **Назначение:** Точка входа (упрощённая версия для быстрого старта)

## Использование на экзамене:
- Данный репозиторий используется **ТОЛЬКО** как источник собственных модулей и компонентов
- Готовые решения задач не копируются
- При решении практической задачи код из `utils.py` может использоваться как библиотека функций
